# 5 Bài tập về Hàm (Function) trong C++

Dưới đây là 5 bài tập đơn giản về hàm. Mỗi bài gồm yêu cầu, ví dụ đầu vào/đầu ra và gợi ý.

---

## Bài 1 — Hàm cộng hai số
Yêu cầu: Viết hàm `int add(int a, int b)` trả về tổng `a + b`. Trong `main` đọc hai số, gọi hàm và in kết quả.

Ví dụ:
Input:
```
3 5
```
Output:
```
8
```

Gợi ý: Trả về `a + b`.

---

## Bài 2 — Hàm hoán đổi (pass-by-reference)
Yêu cầu: Viết hàm `void swapRef(int &x, int &y)` để hoán đổi giá trị hai biến bằng tham chiếu. Trong `main` đọc hai số, in giá trị trước và sau khi gọi hàm.

Ví dụ:
Input:
```
2 7
```
Output:
```
Truoc: 2 7
Sau: 7 2
```

Gợi ý: Dùng biến tạm `tmp` và tham chiếu `&`.

---

## Bài 3 — Hàm tìm giá trị lớn nhất trong ba số
Yêu cầu: Viết hàm `int maxOfThree(int a, int b, int c)` trả về số lớn nhất trong ba số. Trong `main` đọc ba số và in kết quả.

Ví dụ:
Input:
```
4 9 1
```
Output:
```
9
```

Gợi ý: So sánh tuần tự hoặc dùng `std::max`.

---

## Bài 4 — Hàm tính giai thừa
Yêu cầu: Viết hàm `unsigned long long factorial(unsigned int n)` trả về `n!`. Trong `main` đọc `n` (n ≤ 20), gọi hàm và in kết quả.

Ví dụ:
Input:
```
5
```
Output:
```
120
```

Gợi ý: Tính bằng vòng lặp trong thân hàm để tránh đệ quy sâu.

---

## Bài 5 — Hàm kiểm tra số nguyên tố
Yêu cầu: Viết hàm `bool isPrime(int n)` trả về `true` nếu `n` là số nguyên tố, ngược lại `false`. Trong `main` đọc `n` và in `"YES"` hoặc `"NO"`.

Ví dụ:
Input:
```
13
```
Output:
```
YES
```

Gợi ý: Nếu `n < 2` → false; kiểm tra ước từ 2 đến `sqrt(n)` và dừng sớm khi tìm ước.
