# Toán tử trong C++

## 1. Toán tử là gì?

- **Toán tử** là các ký hiệu đặc biệt dùng để thực hiện các phép toán trên biến và giá trị.
- Ví dụ: cộng (`+`), trừ (`-`), nhân (`*`), chia (`/`), so sánh (`==`), gán (`=`),...

---

## 2. Các loại toán tử cơ bản trong C++

### 2.1. Toán tử số học

Sử dụng cho kiểu số (int, float, double):

| Toán tử | Ý nghĩa        | Ví dụ sử dụng    |
|---------|---------------|------------------|
| +       | Cộng          | a + b            |
| -       | Trừ           | a - b            |
| *       | Nhân          | a * b            |
| /       | Chia          | a / b            |
| %       | Chia lấy dư   | a % b (chỉ dùng với số nguyên) |

**Ví dụ:**
```cpp
int a = 7, b = 3;
int tong = a + b;    // 10
int hieu = a - b;    // 4
int tich = a * b;    // 21
int thuong = a / b;  // 2
int du = a % b;      // 1
```

---

### 2.2. Toán tử gán

| Toán tử | Ý nghĩa         | Ví dụ                  |
|---------|----------------|------------------------|
| =       | Gán            | int x = 5;             |
| +=      | Cộng rồi gán   | x += 2;  // x = x + 2; |
| -=      | Trừ rồi gán    | x -= 2;  // x = x - 2; |
| *=      | Nhân rồi gán   | x *= 3;  // x = x * 3; |
| /=      | Chia rồi gán   | x /= 2;  // x = x / 2; |
| %=      | Lấy dư rồi gán | x %= 2;  // x = x % 2; |

---

### 2.3. Toán tử so sánh

Sử dụng để so sánh giá trị của hai biến:

| Toán tử | Ý nghĩa            | Ví dụ      |
|---------|--------------------|------------|
| ==      | Bằng nhau          | a == b     |
| !=      | Khác nhau          | a != b     |
| >       | Lớn hơn            | a > b      |
| <       | Nhỏ hơn            | a < b      |
| >=      | Lớn hơn hoặc bằng  | a >= b     |
| <=      | Nhỏ hơn hoặc bằng  | a <= b     |

Kết quả trả về là giá trị logic: `true` hoặc `false`.

---

### 2.4. Toán tử logic

Dùng với kiểu dữ liệu `bool`:

| Toán tử | Ý nghĩa        | Ví dụ           |
|---------|---------------|-----------------|
| &&      | Và (AND)      | a && b          |
| \|\|    | Hoặc (OR)     | a || b          |
| !       | Phủ định (NOT)| !a              |

**Ví dụ:**
```cpp
bool x = true, y = false;
bool ketQua1 = x && y; // false
bool ketQua2 = x || y; // true
bool ketQua3 = !x;     // false
```

---

### 2.5. Toán tử tăng/giảm

| Toán tử | Ý nghĩa        | Ví dụ sử dụng   |
|---------|---------------|-----------------|
| ++      | Tăng 1 đơn vị | a++; ++a        |
| --      | Giảm 1 đơn vị | a--; --a        |

**Chú ý:**
- `a++`: Tăng giá trị của `a` lên 1, nhưng trả về giá trị trước khi tăng.
- `++a`: Tăng giá trị của `a` lên 1, rồi trả về giá trị sau khi tăng.

---

## 3. Ví dụ tổng hợp

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5, b = 2;
    cout << "a + b = " << a + b << endl;
    cout << "a - b = " << a - b << endl;
    cout << "a * b = " << a * b << endl;
    cout << "a / b = " << a / b << endl;
    cout << "a % b = " << a % b << endl;

    cout << "a == b: " << (a == b) << endl;
    cout << "a > b: " << (a > b) << endl;

    bool ketQua = (a > 3) && (b < 5);
    cout << "Ket qua logic: " << ketQua << endl;

    a++; // a = 6
    cout << "a sau khi tang: " << a << endl;

    return 0;
}
```
