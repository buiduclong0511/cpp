# Biến và kiểu dữ liệu trong C++

## 1. Biến là gì?

- **Biến** là một vùng nhớ được đặt tên dùng để lưu trữ giá trị trong chương trình.
- Mỗi biến sẽ có một tên (identifier), kiểu dữ liệu và giá trị.

**Ví dụ:**
```cpp
int tuoi = 20;
```
Ở đây, `tuoi` là tên biến, kiểu dữ liệu là `int`, giá trị là `20`.

---

## 2. Quy tắc đặt tên biến

- Tên biến phải bắt đầu bằng chữ cái hoặc dấu gạch dưới `_`
- Không được dùng ký tự đặc biệt, không bắt đầu bằng số
- Không trùng với từ khóa của C++ (int, return, if, ...)
- Nên đặt tên biến có ý nghĩa, dễ hiểu

**Ví dụ hợp lệ:** `dem_so`, `_tong`, `soLuong`
**Ví dụ không hợp lệ:** `1tuoi`, `float`, `@name`

---

## 3. Kiểu dữ liệu cơ bản trong C++

| Kiểu dữ liệu | Kích thước | Giá trị lưu trữ | Ví dụ khai báo |
|--------------|------------|-----------------|----------------|
| int          | 4 byte     | Số nguyên       | int a = 10;    |
| float        | 4 byte     | Số thực         | float b = 3.14;|
| double       | 8 byte     | Số thực lớn     | double c = 2.71828183; |
| char         | 1 byte     | Ký tự           | char d = 'A';  |
| bool         | 1 byte     | Đúng/Sai        | bool e = true; |

### Giải thích:
- **int**: Số nguyên, không có phần thập phân.
- **float**: Số thực, có phần thập phân, độ chính xác vừa phải.
- **double**: Số thực, độ chính xác cao (gấp đôi float).
- **char**: Một ký tự (chữ cái, số, ký hiệu).
- **bool**: Giá trị logic (true hoặc false).

---

## 4. Khai báo và gán giá trị cho biến

**Cú pháp:**
```cpp
<kiểu dữ liệu> <tên biến> = <giá trị>;
```

**Ví dụ:**
```cpp
int tuoi = 20;
float diem = 8.5;
char kyTu = 'A';
bool laSinhVien = true;
```

Bạn cũng có thể khai báo nhiều biến cùng kiểu trên một dòng:
```cpp
int a = 5, b = 7, c = 10;
```

---

## 5. Giá trị mặc định của biến

- Biến khai báo nhưng không gán giá trị có thể chứa giá trị rác (không xác định).
- Nên luôn khởi tạo giá trị cho biến khi khai báo.

**Ví dụ:**
```cpp
int x; // giá trị của x chưa xác định
```

---

## 6. Lưu ý khi sử dụng biến và kiểu dữ liệu

- Kiểu dữ liệu quyết định phạm vi giá trị có thể lưu trữ.
- Sử dụng kiểu phù hợp để tiết kiệm bộ nhớ và tránh sai số.
- Không nên gán giá trị kiểu khác cho biến (không khuyến khích ép kiểu khi không cần thiết).

**Ví dụ lỗi:**
```cpp
int tuoi = "hai muoi"; // Sai vì "hai muoi" là chuỗi, không phải số nguyên
```

---

## 7. Ví dụ tổng hợp

```cpp
#include <iostream>
using namespace std;

int main() {
    int tuoi = 18;
    float diemTrungBinh = 7.5;
    char nhom = 'A';
    bool daTotNghiep = false;

    cout << "Tuoi: " << tuoi << endl;
    cout << "Diem TB: " << diemTrungBinh << endl;
    cout << "Nhom: " << nhom << endl;
    cout << "Da tot nghiep: " << daTotNghiep << endl;

    return 0;
}
```

---

## 8. Tổng kết

- Biến là nơi lưu trữ dữ liệu trong chương trình.
- Kiểu dữ liệu xác định loại dữ liệu lưu trữ.
- Luôn khai báo, đặt tên và khởi tạo biến đúng cách để chương trình chạy ổn định.
