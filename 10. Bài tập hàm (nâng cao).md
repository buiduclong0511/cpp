# 5 Bài tập nâng cao về Hàm (Function) trong C++

Những bài sau đây mở rộng từ kiến thức hàm cơ bản — yêu cầu xử lý thêm logic, tổ chức mã tốt hơn hoặc kết hợp nhiều hàm.

---

## Bài 6 — Hàm tính lũy thừa nguyên (integer power) (nâng cao)
Yêu cầu: Viết hàm `long long power(int a, int b)` trả về `a^b` với `b >= 0` (b là số nguyên không âm). Yêu cầu tối ưu: nếu b lớn (ví dụ ≤ 10^9) thì viết phiên bản sử dụng thuật toán lũy thừa nhanh (exponentiation by squaring) và đảm bảo xử lý trường hợp kết quả tràn hợp lý (có thể dùng `long long` và/hoặc kiểm tra tràn).

Ví dụ:
Input:
```
2 10
```
Output:
```
1024
```

Gợi ý: Viết hai phiên bản: (1) đơn giản dùng vòng lặp (dành cho b nhỏ), (2) nhanh bằng cách bình phương lặp.

---

## Bài 7 — Hàm tính Ước chung lớn nhất (GCD) và kiểm tra coprime
Yêu cầu: Viết hàm `int gcd(int a, int b)` trả về ước chung lớn nhất của `a` và `b`.

Ví dụ:
Input:
```
48 18
```
Output:
```
6
```

---

## Bài 8 — Hàm tính tổng chữ số (đệ quy)
Yêu cầu: Viết hàm `int sumDigits(int n)` trả về tổng các chữ số của `n` (n >= 0) sử dụng đệ quy. Trong `main` đọc `n` và in kết quả.

Ví dụ:
Input:
```
12345
```
Output:
```
15
```

Gợi ý: Đệ quy: sumDigits(n) = (n % 10) + sumDigits(n / 10); base case n == 0 -> 0.

---

## Bài 9 — Hàm đảo ngược chữ số của một số (xử lý bằng string và số)
Yêu cầu: Viết hàm:
- `int reverseNumber(int n)` — đảo ngược số bằng toán học (dùng % và /).

Ví dụ:
Input:
```
1230
```
Output:
```
321
```

Gợi ý: Lưu ý bỏ các số 0 ở đầu khi đảo; kiểm tra tính nhất quán giữa hai phương pháp.

---

## Bài 10 — Hàm kiểm tra số đối xứng (palindrome) và tách chức năng
Yêu cầu: Viết hàm `bool isPalindrome(int n)` sử dụng hàm `reverseNumber` từ Bài 9 để kiểm tra. Viết `main` để đọc nhiều số (ví dụ dòng input chứa t số), kiểm tra từng số và in `"YES"`/`"NO"` mỗi dòng. Tách chức năng: một hàm chỉ làm nhiệm vụ đảo, một hàm chỉ làm nhiệm vụ so sánh.

Ví dụ:
Input:
```
3
121
123
1331
```
Output:
```
YES
NO
YES
```

Gợi ý: Tách hàm giúp dễ kiểm thử và tái sử dụng.
