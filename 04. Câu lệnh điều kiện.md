# Câu lệnh điều kiện trong C++

## 1. Khái niệm

- **Câu lệnh điều kiện** cho phép chương trình kiểm tra một điều kiện và thực hiện các hành động khác nhau tùy theo điều kiện đúng hoặc sai.
- Giúp chương trình có thể “ra quyết định” trong quá trình thực thi.

---

## 2. Cấu trúc if - else

### 2.1. Cấu trúc cơ bản

```cpp
if (điều kiện) {
    // Khối lệnh thực hiện khi điều kiện đúng
}
```

### 2.2. if - else

```cpp
if (điều kiện) {
    // Khối lệnh thực hiện khi điều kiện đúng
} else {
    // Khối lệnh thực hiện khi điều kiện sai
}
```

### 2.3. if - else if - else

```cpp
if (điều kiện 1) {
    // Thực hiện nếu điều kiện 1 đúng
} else if (điều kiện 2) {
    // Thực hiện nếu điều kiện 2 đúng
} else {
    // Thực hiện nếu cả hai điều kiện trên đều sai
}
```

---

## 3. Ví dụ sử dụng if - else

```cpp
#include <iostream>
using namespace std;

int main() {
    int diem;
    cout << "Nhap diem: ";
    cin >> diem;

    if (diem >= 5) {
        cout << "Ban da dau!" << endl;
    } else {
        cout << "Ban da rot!" << endl;
    }

    return 0;
}
```

---

## 4. Cấu trúc switch - case

- Dùng để kiểm tra giá trị của một biến nguyên, giúp chương trình rẽ nhánh theo nhiều trường hợp.

**Cú pháp:**
```cpp
switch (bien) {
    case gia_tri_1:
        // Khối lệnh
        break;
    case gia_tri_2:
        // Khối lệnh
        break;
    ...
    default:
        // Khối lệnh mặc định nếu không có trường hợp nào khớp
        break;
}
```

---

### Ví dụ switch - case

```cpp
#include <iostream>
using namespace std;

int main() {
    int thang;
    cout << "Nhap thang (1-12): ";
    cin >> thang;

    switch (thang) {
        case 1:
            cout << "Thang mot" << endl;
            break;
        case 2:
            cout << "Thang hai" << endl;
            break;
        // ... các trường hợp khác
        default:
            cout << "Khong hop le!" << endl;
            break;
    }

    return 0;
}
```

---

## 5. Một số lưu ý khi sử dụng câu lệnh điều kiện

- Điều kiện trong `if` hoặc `else if` phải trả về giá trị logic (`true` hoặc `false`).
- Sử dụng dấu ngoặc nhọn `{}` để bao khối lệnh thực hiện nhiều câu lệnh.
- Không quên dùng `break` trong các trường hợp của `switch` để tránh thực hiện lệnh tiếp theo.

---

## 6. Tổng kết

- Câu lệnh điều kiện giúp chương trình linh hoạt hơn khi xử lý dữ liệu và ra quyết định.
- Cần thành thạo cách sử dụng `if`, `else if`, `else`, và `switch-case` để viết chương trình C++ hiệu quả.
