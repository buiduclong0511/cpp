# Cấu trúc chương trình C++ cơ bản

## 1. Khái niệm chương trình máy tính

- **Chương trình máy tính** là tập hợp các câu lệnh (instructions) được viết bằng một ngôn ngữ lập trình, nhằm điều khiển máy tính thực hiện các thao tác nhất định để giải quyết một vấn đề cụ thể.
- Chương trình thường bao gồm các bước nhập dữ liệu, xử lý dữ liệu và xuất kết quả.
- Các chương trình được biên dịch (compile) và thực thi (run) trên các hệ thống máy tính.

---

## 2. Giới thiệu

Chương trình C++ là một tập hợp các câu lệnh được máy tính thực hiện tuần tự. Để viết được một chương trình C++ hoàn chỉnh, bạn cần nắm vững các thành phần cơ bản sau.

---

## 3. Cấu trúc tổng quát của chương trình C++

```cpp
#include <iostream>

using namespace std;

int main() {
    // Khối lệnh thực thi
    cout << "Hello, World!" << endl;
    return 0;
}
```

### Giải thích từng thành phần:

#### 3.1. `#include <iostream>`

- Đây là câu lệnh tiền xử lý, dùng để khai báo thư viện nhập/xuất dữ liệu (input/output).
- Thư viện này cho phép chương trình sử dụng các đối tượng như `cin`, `cout`.

#### 3.2. `using namespace std;`

- Khai báo sử dụng không gian tên `std` (standard).
- Giúp bạn không phải viết tiền tố `std::` trước các đối tượng chuẩn như `cout`, `cin`.

#### 3.3. `int main()`

- Là hàm chính (entry point) của chương trình C++.
- Khi chương trình bắt đầu chạy, máy tính sẽ thực thi lệnh trong hàm này đầu tiên.
- Hàm `main` luôn trả về một giá trị kiểu nguyên (`int`), thường là `0` nếu chương trình chạy thành công.

#### 3.4. Khối lệnh `{ ... }`

- Các lệnh nằm trong dấu `{}` sẽ được thực thi khi chương trình chạy.

#### 3.5. `cout << "Hello, World!" << endl;`

- Xuất dữ liệu ra màn hình.
- `cout` là đối tượng xuất dữ liệu.
- `<<` là toán tử dùng để truyền dữ liệu vào `cout`.
- `"Hello, World!"` là chuỗi ký tự sẽ được in ra màn hình.
- `endl` dùng để xuống dòng.

#### 3.6. `return 0;`

- Kết thúc chương trình, trả về giá trị 0 cho hệ điều hành, biểu thị chương trình chạy thành công.

---

## 4. Comment trong chương trình

Comment giúp chú thích cho đoạn code mà không ảnh hưởng đến quá trình thực thi chương trình.

- Comment 1 dòng:  
  ```cpp
  // Đây là comment một dòng
  ```
- Comment nhiều dòng:  
  ```cpp
  /* Đây là comment
     nhiều dòng */
  ```

---

## 5. Một số lưu ý khi viết chương trình C++

- Mỗi câu lệnh phải kết thúc bằng dấu `;`
- Tên hàm `main` bắt buộc phải có trong mọi chương trình C++
- Chú ý phân biệt chữ hoa, chữ thường
- Code nên được thụt lề (indent) rõ ràng để dễ đọc và bảo trì

---

## 6. Ví dụ về chương trình C++ đơn giản

```cpp
#include <iostream>
using namespace std;

int main() {
    // In ra màn hình
    cout << "Xin chao ban den voi C++!" << endl;
    return 0;
}
```

---

## 7. Tổng kết

- Chương trình C++ luôn bắt đầu từ hàm `main`
- Sử dụng các thư viện cần thiết với `#include`
- Để chương trình dễ hiểu và bảo trì, nên dùng comment hợp lý
