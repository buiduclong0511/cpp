# Mảng cơ bản trong C++

## 1. Mảng là gì
Mảng là cấu trúc lưu nhiều phần tử cùng kiểu, truy cập bằng chỉ số (index) bắt đầu từ 0.

## 2. Khai báo và khởi tạo
```cpp
int a[5];             // mảng 5 phần tử (giá trị rác nếu local)
int b[5] = {1,2,3};   // còn lại = 0
int c[] = {4,5,6};    // kích thước suy luận = 3
```

## 3. Truy cập và sửa
```cpp
a[0] = 10;
int x = b[2]; // lấy phần tử thứ 3
```
Lưu ý: truy cập ngoài phạm vi (ví dụ a[5]) là undefined behavior.

## 4. Dùng vòng lặp để duyệt
```cpp
for (size_t i = 0; i < 5; ++i) {
    // xử lý a[i]
}

// C++11 range-for (với std::array hoặc nếu biết kích thước cố định)
for (int v : c) {
    // xử lý v
}
```
